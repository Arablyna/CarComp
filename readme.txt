Pour utiliser ce projet, suivez les étapes suivantes :

Importez la base de données "tdw.sql" dans phpMyAdmin en la nommant "tdw".
Assurez-vous que le dossier contenant le projet est nommé "prj".
Remarques :

*Lors de l'ajout d'une actualité, d'un véhicule ou d'une marque, importez les photos à partir du dossier "uploads".
*Les informations d'administration sont les suivantes : nom d'utilisateur "admin", mot de passe "admin".
